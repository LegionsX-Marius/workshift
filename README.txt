Pontaj App - Clock-in/Clock-out
--------------------------------
Functionalități:
- Autentificare: nume + ID + grad + tura (1/2/3)
- Management persoane: doar Director/Vice-Director adaugă/șterg
- Șef Spital: poate vedea orele tuturor (read-only)
- Restul gradelor: văd doar orele proprii
- Ture: flux corect
  * La intrare: introduci ora de START
  * La ieșire: introduci ora de STOP
  * Se calculează automat durata
- Săptămână: Sâmbătă 20:00 → Sâmbăta următoare 18:50
- Design: negru + albastru închis + shadow cian; logo animat în img/logo.png
- Export CSV: persoane, raport săptămânal, orele mele

Cum folosești:
1) Deschide login.html, autentifică-te cu un utilizator existent în listă.
2) Pentru a adăuga utilizatori, autentifică-te ca Director/Vice-Director și mergi pe index.html.
3) După login mergi pe tura.html pentru a începe/termina tura.
4) Folosește butoanele de export CSV pentru rapoarte.
